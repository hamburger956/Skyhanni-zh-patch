## 使用步骤:
- 首先下载保险库补丁(Vault patcher)模组  
- 然后将所有文件拖入游戏主目录(就是有一大堆文件的那个文件夹,其中包含config文件夹、mods文件夹等等)  
- 最后启动游戏！
